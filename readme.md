## any-todo

`npm install` 进行安装

然后分别运行`npm run server` 和 `npm run client` 启动服务端与客户端

#### 简介
使用 <b>react</b>、<b>redux</b> 和 <b>antd</b> 构建的在线管理任务清单Todo应用。前端使用React相关，后端使用了Express + Mongodb

#### 技术栈
* React
* Redux
* Antd
* Express
* Mongodb
* Create-react-app

#### 实现的功能
* 添加删除完成取消
* 用户的登录注册功能
* 存储功能（Mongodb）

后续会继续添加`todo`分享与协作等功能
